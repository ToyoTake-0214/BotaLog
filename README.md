# ボタろぐ

## サービス概要

ボタろぐは、植物の育成記録を写真とメモで残せるアプリです。
植物ごとに水やり頻度を登録し、日々の成長や水やりのタイミングを管理できます。

## 作成背景

植物を育てる中で、「前回いつ水やりをしたか」「どのくらい成長したか」を忘れてしまうことがあります。
そこで、植物ごとに写真とメモを残しながら、育成状況を気軽に管理できるアプリを作成しました。

## 主な機能

* ユーザー登録・ログイン
* 植物の登録・編集・削除
* 育成記録の作成・編集・削除
* 植物画像・育成記録画像のアップロード
* 水やり頻度の登録
* 水やり完了日の記録

## 使用技術

* Ruby 3.3.7
* Ruby on Rails 7.2.3
* PostgreSQL
* Docker / Docker Compose
* Tailwind CSS
