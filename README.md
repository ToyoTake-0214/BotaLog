# ボタろぐ

## サービス概要

ボタろぐは、植物の育成記録を写真とメモで残せるアプリです。  
植物ごとに水やり頻度を登録し、日々の成長や水やりのタイミングを管理できます。

## 公開URL

https://plant-wb5q.onrender.com/

## 作成背景

育てる株が増えていく中で、成長記録や水やりのタイミングを管理するのが少しずつ大変になっていきます。  
また、写真を撮っていても、どの植物のいつの記録なのかが分かりづらくなったり、前回の水やり日を忘れてしまうことがありました。

そこで、植物ごとに写真とメモを残しながら、水やりのタイミングも一緒に管理できるアプリを作成しました。

## 主な機能

* 育成植物の登録
* 植物ごとの育成記録
* 水やり頻度の登録
* 水やり完了日の記録
